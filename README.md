
# Description

[Hestia](https://github.com/lordgape/hestia) API to process CSV for Hestia .

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm start

# watch mode
$ npm run dev
```

## Test

```bash
# unit tests
$ npm test
```
